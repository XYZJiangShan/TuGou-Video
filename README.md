# 🐶 土狗视频下载器 V1.0

一站式视频下载 & 去重处理工具。

## 功能

### 📥 视频下载
- **抖音**：无水印下载（支持分享链接/短链接）
- **快手**：开发中...
- **B站**：开发中...

### 🔄 视频去重（12+种手段）

| 层级 | 手段 | 说明 |
|------|------|------|
| 视觉层 | 亮度/对比度/饱和度/Gamma | 微调色彩参数改变像素值 |
| 视觉层 | 锐化/降噪 | 改变画面细节纹理 |
| 视觉层 | 不可见噪点 | 随机噪点覆盖，人眼不可见 |
| 几何层 | 边缘裁剪 | 四周裁剪1-4%边缘 |
| 几何层 | 水平镜像 | 左右翻转画面 |
| 几何层 | 微旋转 | 0.1-2度旋转 |
| 时间层 | 变速 | 1.02-1.08x加速 |
| 时间层 | 掐头去尾 | 删除首尾帧 |
| 音频层 | 变调 | 微调音高（不易察觉） |
| 音频层 | 变速 | 随视频同步 |
| 元数据层 | MD5修改 | 重编码+随机元数据 |
| 元数据层 | 关键帧随机化 | 随机关键帧间隔 |

### ⚡ 5种预设模式
1. **轻度去重** - 微调色彩+裁剪，画质损失最小
2. **中度去重** - 全面调整，推荐日常使用
3. **深度去重** - 大幅调整+音频变调
4. **镜像翻转模式** - 镜像+色彩微调
5. **极限去重** - 所有手段全开

## 环境要求

- Python 3.10+
- FFmpeg（视频处理核心）
- 依赖库：requests, opencv-python, numpy, Pillow

## 快速开始

### 方法1：双击启动
```
双击 启动土狗下载器.bat
```

### 方法2：命令行
```bash
pip install -r requirements.txt
python app.py
```

### 方法3：仅下载（命令行）
```bash
python -m core.downloader
```

### 方法4：仅去重（命令行）
```bash
python -m core.deduplicator
```
